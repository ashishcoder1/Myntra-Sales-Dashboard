# 📊 Myntra Fashion Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Excel dashboard built using a Myntra fashion dataset. The dashboard analyzes product pricing, discounts, customer ratings, and brand performance to generate meaningful business insights.

The project demonstrates Excel's data analysis and visualization capabilities using Pivot Tables, Pivot Charts, and interactive dashboard elements.

---

## 🎯 Objectives

- Analyze pricing trends across brands
- Identify products with the highest discounts
- Compare brand performance
- Understand customer ratings and review distribution
- Build an interactive dashboard for business decision-making

---

## 📂 Dataset

The dataset contains information about Myntra fashion products, including:

- Brand Name
- Product Description
- Selling Price
- MRP
- Discount %
- Discount Amount
- Ratings
- Rating Category
- Number of Ratings
